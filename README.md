## 声明

- 本项目为本人学习用的开源项目，仅供学习交流使用。项目中使用的音乐数据及 API 均来自第三方，版权归属于网易云音乐及原作者。请勿用于任何商业用途。
- 暂不支持播放本地音乐。
- 不提供后端服务，如有需要请自行搭建。

- TypeScript
- Rust
- Tauri
- React
- TailwindCSS
- shadcn/ui
- Zustand
